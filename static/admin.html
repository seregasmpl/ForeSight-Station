ИНСТРУКЦИЯ ДЛЯ УЧИТЕЛЯ

  1. Перед началом — нажмите СОЗДАТЬ СЕССИИ. Появятся 8 карточек (4 темы × 2 роли).
  2. Раздайте коды командам. Каждая команда вводит свой код на главной странице (например, ЛУНА-01).
  3. Во время игры — команды задают вопросы ИИ-консультанту. Он отвечает по 7 разделам: позиция, аргументы, прогнозы, риски, речь для дебатов, вопросы оппонентам, новость из 2100 года.
  4. Карточки обновляются каждые 5 секунд. Зелёная точка = онлайн, жёлтая = ожидает, красная = отключена.
  5. Нажмите на карточку — увидите историю вопросов команды.
  6. СБРОСИТЬ ВСЕ — обнуляет все сессии и историю (нельзя отменить!).
